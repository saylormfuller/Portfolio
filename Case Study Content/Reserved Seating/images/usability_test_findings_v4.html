Key findings
High task completion
Most participants completed the workflow once they found it, indicating core functionality is usable.
Poor feature discoverability
Reserved Seating and Seat Manager were consistently hard to locate, especially within nested dropdown menus.
Navigation uncertainty
Many users explored multiple tabs before finding the correct path — IA did not match user expectations.
Lack of system feedback
Participants were unsure whether seat selections were saved — the interface provided little post-action confirmation.
Prototype limitations surfaced concerns
Pre-completed tasks and unsaved states introduced confusion, reinforcing the need for clear system feedback.
